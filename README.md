## Usage

Integrating VecMath into your project is straightforward. Below is a practical example of how to perform vector transformations:

```cpp
#include <iostream>
#include "vec2.hpp"

int main() {
    // Initialize vectors
    Vector2 position(10, 15);
    Vector2 velocity(2, 3);

    // Perform vector addition using overloaded operators
    position += velocity;

    // Scale the vector
    position.scale(2);

    // Normalize and get length
    float length = position.getLength();
    Vector2 direction = position.normalized();

    std::cout << "Magnitude: " << length << std::endl;

    return 0;
}
```
